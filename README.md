# landing-deploy

An infrastructure deployment with Terraform, Ansible and self-deployed Kubernetes on Google Cloud Platform following GitOps practices.

## Requirements

- `terraform`[^1],
- `ansible`[^2],
- `gcloud`[^3],
- `kubectl`[^4] (optional).
